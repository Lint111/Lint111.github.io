# Web of Lies: Spreading Misinformation

## Overview
"Web of Lies: Spreading Misinformation" is an engaging game that challenges players to navigate through a web of false information and uncover the truth. Players must use critical thinking and strategy to differentiate between credible sources and misleading narratives.

## Features
- **Interactive Gameplay**: Players engage with various scenarios that test their ability to identify misinformation.
- **Educational Elements**: The game incorporates real-world examples of misinformation, providing players with insights into media literacy.
- **Dynamic Storytelling**: Choices made by players affect the outcome of the game, encouraging multiple playthroughs.

## Technologies Used
- **Game Engine**: Unity
- **Programming Languages**: C#, C++
- **Graphics API**: Vulkan
